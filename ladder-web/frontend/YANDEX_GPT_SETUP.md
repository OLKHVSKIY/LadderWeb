# Настройка Yandex GPT API

## Как получить API ключ от Yandex GPT

1. **Перейдите в Yandex Cloud Console**
   - Откройте https://console.cloud.yandex.ru/
   - Войдите в свой аккаунт

2. **Создайте сервисный аккаунт**
   - Перейдите в раздел "Identity and Access Management" (IAM)
   - Нажмите "Создать сервисный аккаунт"
   - Укажите имя (например, "ladder-ai")
   - Нажмите "Создать"

3. **Назначьте роли сервисному аккаунту**
   - Выберите созданный сервисный аккаунт
   - Перейдите на вкладку "Роли"
   - Нажмите "Назначить роль"
   - Выберите роль: `ai.languageModels.user` или `ai.foundationModels.user`
   - Сохраните

4. **Создайте API ключ**
   - В разделе IAM выберите ваш сервисный аккаунт
   - Перейдите на вкладку "API-ключи"
   - Нажмите "Создать API-ключ"
   - **ВАЖНО**: Скопируйте ключ сразу! Он будет показан только один раз
   - Ключ будет выглядеть как: `AQVNxxxxxxxxxxxxx`

5. **Вставьте ключ в конфигурацию**
   - Откройте файл: `/frontend/assets/js/config/yandex-gpt-config.js`
   - Вставьте ваш API ключ в поле `API_KEY: 'ваш_ключ_здесь'`
   - Сохраните файл

## Использование API Gateway

Если вы используете Yandex API Gateway (как показано в вашем примере), убедитесь, что:
- В конфигурации указан правильный `API_GATEWAY_URL`
- API Gateway настроен для работы с Yandex GPT
- В API Gateway указан правильный метод аутентификации (Api-Key)

## Проверка работы

После настройки API ключа:
1. Откройте страницу чата: http://localhost:8080/public/chat.html
2. Напишите сообщение
3. Если все настроено правильно, вы получите ответ от Yandex GPT
